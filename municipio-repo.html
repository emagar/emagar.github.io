Description

Welcome! This web page simplifies downloading of the datasets described in the research note that can be found here. The page links to datasets of vote returns for Mexican municipal government elections over recent decades. These datasets belong in my Recent Mexican Election Vote Returns repository at https://github.com/emagar/elecRetrns, where they are maintained and expanded, along with many more datasets.

Links to the municipal elections datasets

To access the municipal elections datasets, click the desired link from the table and then either click the 'view raw' or the download icon. See the research note for details on the contents.
