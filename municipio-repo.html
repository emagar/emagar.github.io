Description

Welcome! This webpage has direct links to the datasets described in the research note that can be found here.

The datasets contain vote returns for Mexican municipal government elections over recent decades. These datasets are archived and maintained in my Recent Mexican Election Vote Returns repository that can be found here. Given the size of that repository — which includes electoral data for other state and federal elections at different levels of aggregation, along primary sources, electoral laws, ancillary files, and much more — offering permalinks to the files listed in the research note substantively eases their location, inspection, and downloading.
