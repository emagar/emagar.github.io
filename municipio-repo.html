Description

Welcome. This web page allows easy download of the datasets described in the research note that can be found here. These are datasets of vote returns for Mexican municipal government elections over recent decades.

All datasets linked below belong in my Recent Mexican Election Vote Returns repository at https://github.com/emagar/elecRetrns.
